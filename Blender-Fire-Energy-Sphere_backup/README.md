# Blender-Fire-Energy-Sphere

A energy sphere scene created in Blender using Shader Nodes.

# Overview

This project is a Blender scene designed to create a fiery energy sphere visual.

It uses Shader Nodes to generate flowing, flame-like patterns inside a sphere, creating a powerful and dynamic energy effect.

The scene can be freely modified and used for learning shader workflows or for creating similar visual styles.

## Features

- Shader-based energy effect  
- Customizable colors and density  
- Useful for learning shader workflows and visual experimentation  

# Software Requirement

Blender Version 3.6 LTS

# Rendering Settings

Engine: Cycles

Sampling: 1024

Denoising: Enabled

Original Inspiration: Ducky 3D  

Tutorial: https://www.youtube.com/watch?v=rtPzXrZ23k0&t=192s

# Blender-Fire-Energy-Sphere

シェーダーノードを使用して作成された、炎のエネルギー球のBlenderシーンです。

# 概要

このプロジェクトは、球体内部に流れる炎のような動きを持つエネルギー表現を作成するためのBlenderシーンです。

シェーダーノードを使用して、流動する炎状のパターンと発光を組み合わせたダイナミックなビジュアルを生成します。

シェーダー構成の学習や、ビジュアル制作のベースとして自由に改変して使用できます。

## 特徴

- シェーダーノードによるエネルギー生成  
- 色や密度の調整が可能  
- シェーダーワークフローの学習やビジュアル実験に適しています  

# ソフトウェア要件

Blender バージョン3.6 LTS
